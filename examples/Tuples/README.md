
# Tuples

This example briefly shows different uses of tuples in Roc.
A tuple is an ordered sequence of elements where the elements may have different types.

## Output

```
% roc run main.roc
First is: A String,
Second is: true, 
Third is: 15000000.
You also have some pears.
```
